Publish

上传到 GitHub 后就能访问

1

新建仓库，仓库名必须是 你的GitHub用户名.github.io。

2

把这个文件夹里的所有文件上传到仓库根目录。

3

进入 GitHub 的 Settings → Pages，选择从主分支部署。

4

几分钟后访问 https://你的GitHub用户名.github.io。
